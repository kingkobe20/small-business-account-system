# Small Business Account System

## Description
The Small Business Account System is a simple Python program designed to help small business owners record daily sales and expenses and automatically calculate profit or loss.  
It serves as a practical tool for managing finances and learning basic accounting concepts in BMS.

---

## Features
- Record daily sales  
- Record daily expenses  
- Automatically calculate profit or loss  
- Display a clear daily business report  

---

## Tools Used
- Python 3.x  
- Any code editor (VS Code, PyCharm, IDLE)  

---

## How to Run
1. Clone or download the repository.  
2. Open the project folder.  
3. Run `main.py` in your Python editor or terminal.  
4. Enter sales and expenses when prompted.  
5. View the generated daily report.
